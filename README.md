# 🎮 Aduk Games – Demo Platform

Welcome to the **Aduk Games demo** – a fully functional, single‑page web application showcasing 25+ playable games (slots, card games, arcade, skill) with a shared balance, daily challenge, leaderboard, and more.  
This demo runs entirely in your browser; no backend or installation needed.

## ✨ Features

- **25+ playable games** – Slot Machine, Crash, Dice, Roulette, Keno, Scratch Card, Wheel of Fortune, Blackjack, Poker, Baccarat, Teen Patti, Video Poker, War, Solitaire, Tic‑Tac‑Toe, Chess, Checkers, Connect Four, Backgammon, Memory Match, 2D Soccer, Horse Racing, Progressive Jackpot, Lottery, Rush Hour (simplified).
- **Game list with search & category filter** – quickly find games by name or type (slots, card, skill, arcade, lottery).
- **Global balance** – shared across all games, updates in real time.
- **Dashboard** – wallet, game stats, recent transactions.
- **Daily Challenge** – spin a wheel once per day for bonus balance (uses `localStorage`).
- **Leaderboard** – static preview of top players.
- **Avatar selector** – choose an emoji avatar (saved in `localStorage`).
- **Social sharing** – share your current balance on Twitter, Facebook, WhatsApp, or via the Web Share API.
- **Responsible Gambling page** – information and placeholder tools.
- **Feedback & report forms** – open your email client with pre‑filled details.
- **Tutorial popups** – show game instructions on first visit (also saved).

## 🛠️ Built With

- **HTML5** – semantic structure.
- **CSS3** – custom properties, grid, flex, animations, glass‑morphism.
- **JavaScript (ES6)** – classes, `localStorage`, `requestAnimationFrame`, event handling.
- **No frameworks or build tools** – pure vanilla code, runs anywhere.

## 🚀 How to Run

1. **Download or clone** this repository.
2. Open the folder and double‑click `index.html` – it will launch in your default browser.
3. That's it! All features work offline.
